| 캐스팅 유동성 (주인공 미정) | ✅ | ❌ | ❌ | △ | ❌ |
| 한 세션 유한 (인생 끝) | ✅ | ✅ | ✅ | ❌ | ❌ |

**우리만의 진짜 차별화 두 가지:**

1. **Pre-Story Cradle (전사 요람)** — 사용자가 드라마 시작 전 시대를 살며 캐릭터성을 형성하고, T-0 시점에 누적 패턴이 *어떤 인물이 될지*를 자동 결정. 주인공도 될 수 있고 조연도 될 수 있음.

2. **드라마 IP 라이선스 없이 정서를 가져옴** — 특정 드라마를 가리키지 않으면서, 그 시대 정서를 닮은 시나리오로 K-드라마 팬덤이 즉시 알아채는 구조.

## 3. 핵심 사용자 시나리오 (한 문장씩)

1. 외국인 K-드라마 팬이 TikTok에서 우리 게임 결과 카드를 봄
2. 회원가입 (Google/Apple, 30초)
3. 나이/국가/관심 K-드라마 입력 (1분)
4. AI가 시나리오 5개 추천, 그 중 *Mr. Sunshine 정서* 선택
5. **Cradle 단계 시작** (시나리오에 따라 어린 시절 본인 또는 부모 양육)
6. 누적된 결정이 캐릭터성 형성
7. **T-0 시점에 캐스팅 결정** — 누적 패턴이 *주인공 격* 또는 *조연* 또는 *목격자* 결정
8. Main Story 단계 (드라마와 동행 또는 갈라진 길)
9. 인생 종료, AI가 회고 narrative 작성
10. 결과 카드 자동 생성 → SNS 공유
11. 마이페이지에 결말 컬렉션 1개 추가 (*"발견 1/200"*)
12. *"다른 인생 살아보기"* → 새 시나리오 또는 같은 시나리오 다른 캐스팅

## 4. 기술 스택 (한 줄 요약)

```
Frontend: Next.js 14 (App Router) + Tailwind CSS + shadcn/ui
Backend: Next.js API Routes + Firebase Functions
Database: Firestore (asia-northeast3)
Auth: Firebase Auth (Google, Apple, Email)
AI: Google Gemini 2.0 Flash (routine), Gemini 2.0 Pro (complex)
Image Gen: Stable Diffusion API (결과 카드 일러스트)
Hosting: Vercel (Next.js), Firebase Hosting (정적)
Payment: Stripe (글로벌) + 토스페이먼츠 (한국)
Maps: Google Maps API (위치 카드)
i18n: next-intl (한국어/영어 동시)
```

## 8. 용어 정리

본 패키지에서 일관되게 사용하는 용어:

- **시나리오 (Scenario)**: 한 시대 + 한 캐스팅 풀의 묶음. 예: *Mr. Sunshine 정서*
- **Pre-Story Cradle (전사 요람)**: 드라마 시작 전 시대를 사는 단계 (T-7년 ~ T-0년)
- **T-0 모먼트**: 드라마가 시작되는 시점, 사용자 캐스팅이 자동 결정되는 순간
- **캐스팅 풀 (Casting Pool)**: 한 시나리오 안에서 사용자가 될 수 있는 인물 후보 (5명)
- **챕터 (Chapter)**: 1년 = 1챕터. 한 인생의 챕터 수는 시나리오 cradleConfig + mainStoryEndAge에 따라 가변 (보통 8~14챕터)
- **이벤트 (Event)**: 챕터 안의 단위 결정 모먼트. 챕터당 5~7개
- **게이트 (Gate)**: 이벤트의 평가 방식 5가지 (지식/검색/철학/위치/창작)
- **명장면 (Iconic Moment)**: 드라마의 핵심 장면 정서를 담은 특별 이벤트
- **Quality**: 캐릭터의 누적 변수 (스탯, 인연, 변수 플래그). Failbetter 패턴.
- **결말 (Ending)**: Main Story 종료 시점에 결정되는 인생의 종착점
- **결말 카드 (Ending Card)**: 결말 narrative + 일러스트 + 통계가 담긴 SNS 공유 가능 이미지
- **히어로 카드 (Hero Card)**: 인생 진행 중 사용 가능한 특수 능력 카드
- **위치 카드 (Location Card)**: 실제 장소 방문 시 획득 가능한 보너스 카드
