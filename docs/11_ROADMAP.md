## 10. 핵심 KPI 추적

매주 모니터링:

```
획득 (Acquisition)
- 일일 신규 가입
- 채널별 가입 (TikTok / 검색 / 친구 초대)
- CAC (Customer Acquisition Cost)

활성도 (Engagement)
- DAU / MAU
- 세션당 평균 시간
- 인생 완주율 (시작 → Main Story 종료)
- T-0 도달률
- 자유 입력 사용률

수익 (Revenue)
- 결제 전환율
- ARPU (사용자당 평균 매출)
- 구독 vs 카드팩 비중
- LTV (Customer Lifetime Value)

콘텐츠 성과
- 시나리오별 완주율
- 결말 발견 분포
- 명장면별 선택 분포
- 위치 카드 인증률

품질
- 모더레이션 거부율
- 에러율
- 버그 신고율
- App Store / Google Play 평점
```
