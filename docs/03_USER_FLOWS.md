# 03. 사용자 여정 (User Flows)

이 문서는 사용자가 K-Drama Life를 처음 만나는 순간부터 한 인생을 끝내고 컬렉션에 누적시키는 전 과정을 정의한다.

---

## 1. 진입 시나리오 4가지

사용자가 우리 서비스에 도달하는 경로:

### 1.1 진입 1: TikTok 결과 카드 → 직접 진입

```
TikTok 영상에서 친구 결과 카드를 봄
   ↓
영상 설명 또는 카드의 CTA: "kdramalife.app"
   ↓
모바일 브라우저로 URL 입력 또는 링크 클릭
   ↓
[랜딩 페이지]
```

### 1.2 진입 2: 인플루언서 협찬 후킹

```
유튜브 K-드라마 채널에서 게임 플레이 영상 봄
   ↓
영상 설명에 추천 링크 + 할인 코드
   ↓
[랜딩 페이지에 코드 적용된 상태로 진입]
```

### 1.3 진입 3: 검색 SEO

```
구글에서 "Joseon Dynasty life simulator" 검색
   ↓
우리 SEO 콘텐츠 도달
   ↓
[블로그 → 랜딩 → 게임]
```

### 1.4 진입 4: 친구 초대

```
카카오톡으로 친구가 결과 카드 공유
   ↓
"이거 한번 해봐" + 링크
   ↓
[랜딩 → 회원가입 → 첫 시나리오]
```

## 2. 메인 진입 흐름 (가장 중요한 흐름)

### 2.1 0초 — 랜딩 페이지

**모바일 우선 디자인.**

```
┌───────────────────────────────────────┐
│                                       │
│   [배경 영상: 한국 시대 분위기 콜라주]    │  ← 5초 루프
│                                       │
│       K-DRAMA LIFE                    │
│                                       │
│   "한 시대의 한 사람을 살아본다"          │
│   "당신의 선택이 결정한다"               │
│                                       │
│   ┌─────────────────────────────┐    │
│   │   ▶ 첫 인생 시작하기           │    │  ← CTA 버튼 (큰 사이즈)
│   └─────────────────────────────┘    │
│                                       │
│   이미 계정이 있나요? [로그인]            │
│                                       │
└───────────────────────────────────────┘

[스크롤 다운하면 추가 정보]

   - 30개 시나리오 (조선 궁중부터 1980 골목까지)
   - AI가 매번 새로 써주는 인생
   - 1억 명의 결말 발견 가능성
   - 무료로 시작
```

**디자인 원칙:**
- 첫 화면에 *"첫 인생 시작하기"* 버튼이 fold 위에
- 회원가입 없이 바로 첫 시나리오 살짝 맛볼 수 있는 게 더 좋음 (Phase 2에서 도입)
- Phase 1에선 회원가입 필수

### 2.2 30초 — 회원가입

**최소한의 마찰.**

```
┌───────────────────────────────────────┐
│   회원가입 / 로그인                     │
│                                       │
│   [Google로 시작하기]                  │
│   [Apple로 시작하기]                   │
│   [이메일로 시작하기]                   │
│                                       │
│   [한국어]  [English]                  │  ← 언어 선택
│                                       │
└───────────────────────────────────────┘
```

이메일 회원가입 시:
- 이메일 + 비밀번호만 (이름 필수 X)
- 이메일 인증은 Phase 2에서 (Phase 1은 즉시 가입)

- 입력 데이터는 Firestore `users/{userId}/profile`에 저장
- 추천 시나리오 알고리즘에 활용

- 다음 이벤트로

명장면이나 특정 이벤트 종료 시 자동 트리거:

### 5.1 한국 거주자

```
┌───────────────────────────────────────┐
│                                       │
│   🎴 위치 보너스 가능!                  │
│                                       │

│                                       │
```

- 사진 촬영 + GPS 인증
- 카드 획득 애니메이션
- 게임 내 인벤토리에 추가
