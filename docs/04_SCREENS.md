# 04. 화면 명세

각 화면별 구성요소, 인터랙션, 상태 관리 명세.

---

## 0. 디자인 컨셉 — *한낮의 한지* (Hanji at Noon)

### 0.1 한 줄

> *"사용자는 UI를 '조작'하는 것이 아니라, 한 장의 한지 위에 인생을 '읽고 넘긴다'."*

### 0.2 컨셉 키워드

- 시집 같은 경험
- 따뜻한 한낮
- 한국적 아름다움
- 읽고, 선택하고, 기억되는 인생

### 0.3 무드

따뜻한 한낮의 빛, 종이의 질감, 한국적인 아름다움, 책을 읽는 듯한 경험.

---

## 0.4 컬러 시스템

### Base (배경)

| 역할 | HEX | 비고 |
|---|---|---|
| 메인 배경 | `#F5EFE0` | 한지 베이지 |
| 카드 배경 | `#FBF7ED` | 한 단계 밝은 한지 |
| 그림자 | `rgba(0,0,0,0.04)` | 매우 약한 paper shadow |

### Text

| 역할 | HEX | 비고 |
|---|---|---|
| 본문 | `#1A1614` | 먹 검정 |
| 보조 | `#5B514B` | 짙은 갈색 |
| 캡션 | `#8A7F77` | 회갈색 |

### Accent (상황별 제한적 사용)

| 역할 | HEX | 사용처 |
|---|---|---|
| 단풍 | `#C9302C` | 결정의 순간, 중요 CTA |
| 옥색 | `#5C8E76` | 사극 시대 강조, 평온 |
| 황금 | `#D4AF37` | 귀함, 결말 카드 강조 |

**원칙:** Accent는 한 화면에 1~2개 미만. 항상 한지 베이스 위에서 *작은 도장처럼* 사용.

## 0.5 타이포그래피

| 종류 | 한국어 | 영어 | 사용처 |
|---|---|---|---|
| Title (Serif) | Noto Serif KR | Noto Serif | 시대극 강조, 결말 narrative |
| Body (Sans) | Pretendard Variable | Pretendard / Inter | 본문, 선택지 |
| Caption | Pretendard | Pretendard / Inter | 보조 정보, 메타 |

**규칙**
- 본문 줄간격: 1.7 (호흡 있게)
- 결말/T-0 같은 결정 모먼트는 Serif로 무게감
- 일반 인터랙션은 Sans로 깔끔하게

## 0.6 Spacing System (모바일 기준)

| 토큰 | 값 | 사용처 |
|---|---|---|
| 기본 단위 | `4px` | 모든 간격은 4의 배수 |
| 카드 간격 | `16px` | 카드와 카드 사이 |
| 좌우 패딩 | `20px` | 화면 좌우 여백 |
| 섹션 간격 | `28~32px` | 큰 영역 구분 |

## 0.7 Paper Layer Structure

화면은 *세 겹의 종이*처럼 구성된다:

```
01. 배경 (한지 텍스처)         ← 가장 깊은 층
02. 카드 (아주 약한 그림자)    ← 중간
03. 인터랙션 요소 (거의 flat)   ← 표면
```

**금지:**
- 강한 그림자 (`box-shadow > 4px blur`)
- 네온 / 블루 계열 색상
- 과한 그라데이션

## 0.8 UI Components 표준

### 버튼

| 상태 | 스타일 |
|---|---|
| 기본 | 한지 베이지 배경 + 먹 검정 텍스트 + 미세 outline |
| 선택 (CTA) | 단풍 색 배경 + 흰색 텍스트, 작은 단풍잎 아이콘 |
| 비활성 | 한지 + 회갈색 텍스트, 50% 투명 |

### 카드 (기본)

```
┌────────────────────────┐
│ [이미지]                │
│                        │
│  1988                  │
│  (Reply 1988 정서)     │
│  그 시절, 우리의...      │
│                        │
│                    >   │
└────────────────────────┘
```

- 라운드 코너 12px
- 매우 약한 paper_shadow

### 진행 바

- 한지 배경 트랙 + 단풍 색 진행 (얇은 선)
- 좌측: 현재 챕터 라벨 / 우측: 퍼센트

### 스탯 컴포넌트

- 6각형 라인 아이콘 (지성, 감성, 용기, 매력, 체력, 운)
- 옆에 숫자 라벨
- 변화 시 약 단풍색 빛이 0.5초 깜박

### 상단바 / 탭 메뉴

- 상단: `< CHAPTER 3 ≡`
- 하단 탭 (마이페이지): 스토리 / 스탯 / 기록 / 컬렉션 / 설정
- 모두 라인 아이콘, 한국적 따뜻한 느낌

### 아이콘 스타일

- 손글씨 느낌의 라인 아이콘
- 두께 1.5px
- 라운드 끝선
- 한국적, 따뜻한 결

## 0.9 Interaction & Motion

| 동작 | 효과 | 시간 |
|---|---|---|
| 화면 전환 | 페이드 인 + 아래로 10px 이동 | 300ms |
| 선택 시 | 배경 10% 어두워짐 + 왼쪽 포인트 라인 등장 | 150ms |
| 엔딩 등장 | 페이드 인 | 300ms |
| 버튼 / 카드 / 스탯 변화 | 페이드 | 200ms |
| T-0 모먼트 | 영화적 5초 시퀀스 (전체 화면 어둠 → 해 뜸) | 5000ms |

**규칙:** 모든 모션은 `ease-out`. 빠른 애니메이션 / 튕기는 효과 금지.

## 0.10 DO & DON'T

**DO**
- 따뜻한 톤
- 여백 활용 (한지 위 먹의 여백)
- 텍스트 중심
- 전통적 요소를 작게

**DON'T**
- 강한 그림자
- 네온 / 블루 계열
- 과한 장식
- 빠른 애니메이션

## 2. S01: 랜딩 페이지

### 2.1 목적

- 첫 방문자를 회원가입으로 유도
- 서비스 컨셉을 30초 안에 전달
- 모바일 fold 위에서 결정 가능하도록

### 2.2 구성요소

```
[Hero Section]
- 배경 영상 (5초 루프, 자동 재생, 음소거)
  - 한국 시대 정서 콜라주: 한복, 궁궐, 1900년 한성, 1980 골목
- 로고 (중앙 상단)
- 헤드라인: "한 시대의 한 사람을 살아본다"
- 서브: "당신의 선택이 결정한다"
- CTA 버튼: "▶ 첫 인생 시작하기" (Primary, 큰 사이즈)
- 보조 링크: "이미 계정이 있나요? 로그인"

[Below Fold - 서비스 소개]
- "30개 시나리오, 1억 명의 결말 가능성" (numerical hook)
- 시나리오 일부 카드 노출 (3~4개, 가로 스크롤)
- 결과 카드 갤러리 (커뮤니티 베스트, 추천만 큐레이션)
- "How it works" 3단계 인포그래픽
  1. 시대를 선택하세요
  2. 그 시대 안에서 살아가세요
  3. 당신만의 결말을 만나세요
- FAQ
- Footer
```

### 2.3 동작

- CTA 클릭 → `S02` 회원가입
- 로그인 클릭 → `S03` 로그인
- 비회원도 시나리오 카드 미리보기 가능 (회원가입 유도용)

### 2.4 SEO

- 메타 태그: 한국어, 영어 동시 생성
- OG 이미지: 결과 카드 디자인
- Schema.org: VideoGame markup

## 5. S05: 시나리오 추천

### 5.1 알고리즘

사용자 프로필을 토대로 5개 시나리오 추천. 다양성 보장:

```
규칙:
1. 사용자 선호 장르 1개 이상 매칭하는 시나리오 우선
2. 시대 다양성 (5개 중 사극 1, 근현대 1, 현대 1, 무거운 톤 1, 가벼운 톤 1)
3. 인기도 가중치 (플레이 횟수 많은 것 우선)
4. 사용자 연령에 적합한 톤 (13~17세는 무거운 시나리오 노출 X)
```

### 5.2 화면 구성

```
[헤더]
- "당신을 위해 골랐어요"
- 부제: "어떤 인생을 살아보시겠어요?"

[5개 카드 (세로 스크롤)]

각 카드:
┌─────────────────────────────────────┐
│ [16:9 일러스트 - 시대 분위기]         │
│                                     │
│ [장르 태그]  [무게 표시: ●●○○]       │
│                                     │
│ MR. SUNSHINE 정서                   │
│ 1894~1907 한성                      │
│                                     │
│ "격동의 시대, 어느 자리에 있을 것인가" │
│                                     │
│ ⭐ 4.7 │ 1,247명 플레이              │
│ 결말 47개 │ 평균 50분                │
│                                     │
│ [▶ 이 인생 시작하기]                  │
└─────────────────────────────────────┘

[하단 링크]
- "전체 시나리오 보기" → S06
```

### 5.3 일러스트

- Stable Diffusion API + 한국 미술 스타일 finetuned 모델
- 시나리오마다 1장 (랜딩 카드용)
- 챕터별 추가 일러스트는 Phase 2

### 8.1 구성

### 15.1 한지 텍스처 처리

배경에 미세한 한지 결을 표현. CSS:

```css
body {
  background-color: #F5EFE0;
  background-image: url('/textures/hanji-noise-subtle.png');
  background-size: 200px;
  background-blend-mode: multiply;
  opacity-image: 0.06;  /* 매우 은은하게 */
}
```

또는 SVG noise filter로 처리. 텍스처는 *느낌으로만* 존재하고 인지적으로 거슬리지 않게.

### 15.2 단풍잎 / 한국적 장식

- 카드 모서리, 알림 카드, 선택 강조 등에 작은 단풍잎 또는 도장(인장) 모티프
- SVG 라인 아이콘으로 작게 사용
- 화면당 1~2개 미만 (과한 장식 금지)

### 15.3 다크 모드?

**Phase 1: 다크 모드 미지원.** 한낮의 한지 컨셉 자체가 밝은 톤이 정체성이므로 다크 모드는 컨셉을 깬다.

향후 사용자 요구가 강하면 *"황혼 모드"* (옅은 자주·황혼 톤)로 별도 기획 가능.
